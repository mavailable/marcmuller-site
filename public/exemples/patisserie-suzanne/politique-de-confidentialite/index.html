Donnees collectees

Nous collectons uniquement les donnees que vous fournissez volontairement via nos formulaires :

  • Formulaire de contact : nom, adresse email, telephone (optionnel), message
  • Formulaire de commande sur mesure : nom, adresse email, telephone, type d'evenement, date, nombre de parts, budget indicatif, message

Aucune donnee n'est collectee automatiquement lors de votre navigation sur le site.

Finalite du traitement

Vos donnees sont utilisees exclusivement pour :

  • Repondre a vos demandes de contact
  • Traiter vos demandes de commande sur mesure (devis, suivi)

Vos donnees ne sont jamais utilisees a des fins commerciales, marketing ou de prospection.

Base legale

Le traitement repose sur votre consentement, exprime par la soumission volontaire du formulaire (article 6.1.a du RGPD).

Duree de conservation

Les donnees collectees sont conservees pendant une duree maximale de 3 ans a compter du dernier echange. Elles sont supprimees a l'issue de ce delai ou sur demande de votre part.
