## Comment utiliser

1. Ouvrir `linkedin-assets.html` dans un navigateur web
2. Cliquer sur "Télécharger PNG" pour générer les visuels en PNG haute résolution
3. Les fichiers sont téléchargés automatiquement (1200×627 ou 1080×1080/1350 selon le format)
4. Utiliser sur LinkedIn selon le contexte (bannière, publication, annonce)
