✅ PROCHAINES ÉTAPES
─────────────────────────────────────────────────────────────────────────────

1. INSTALL FONT SATOSHI
   → Télécharger Satoshi-Variable.woff2
   → Placer dans /public/fonts/
   → Les @font-face charge automatiquement

2. BUILD & TEST
   npm run build
   npm run dev
   → Vérifier qu'aucune erreur
   → Tester animations, responsive, animations

3. METTRE À JOUR LES PAGES
   → Utiliser nouveaux composants (Button, Card, SectionTitle, CTASection)
   → Ajouter iconSvg aux Card
   → Ajouter badge aux SectionTitle
   → Utiliser reassurance sur CTASection

4. OPTIMISER ICONS SVG
   → Créer les icons design, performance, maintenance, SEO, support, etc.
   → Utiliser la prop iconSvg (string HTML inline)

5. TESTER TOUT
   → Chrome, Firefox, Safari, Edge
   → Mobile, tablet, desktop
   → Keyboard navigation
   → Screen reader (NVDA, JAWS)

6. DÉPLOYER
   → npm run build
   → Copier dist/ sur serveur
